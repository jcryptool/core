ADFGVX Usage

This page is about using the user interface for the ADFGVX cipher. For general information about this algorithm, see: ADFGVX cipher in the cryptology manual.

Below you will find explanation of what the different "groups" of options shown in the wizard mean, and what to fill in. In the ADFGVX wizard, the "Key" (two of them) and "Operation" groups are mandatory to take care of in most cases.

Operation – encryption or decryption

First, choose, whether you want to en- or decrypt the text in this dialog. As this is a symmetric cipher, a text encrypted with two certain keys can be decrypted to the original text, using the same two keys.

Alphabet

The alphabet, over which a cryptographic operation (like en- or decryption) is executed, holds all the characters that this operation will accept and process. Use the "show alphabet" button to see, which characters the selected alphabet contains.

For example, if you choose an alphabet that holds only UPPERCASE characters (like "Upper Latin (A-Z)"), lowercase characters will not be encrypted. With the checkbox "filter non-alphabet characters", you can choose whether just to neglect (unchecked), or to completely delete those characters, which would be all lowercase characters in the example.

Pre-operation text transformation

If you would like to change the input text before you apply the algorithm, you can check this box, and you will be able to select from a range of text-altering operations after completing this wizard page. For example, you can choose to delete all blanks for increased security.

Pre-operation text transformation should only be used when encrypting a text. When used with decryption, there will be information loss or unreadable text in most cases.

Key

The ADFGVX cipher needs two keys. The first key is responsible for substitution of characters through a key-generated matrix that is shown to the right. The second key is form transposition, which swaps positions of characters around the text. Only keys are allowed, which only contain characters from the alphabet selected above.

After finishing the wizard, you should be able to see your en- or decrypted text in the newly opened editor.
