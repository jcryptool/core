Allgemeine Einstellungen

Die Einstellungen für die XML Sicherheit sind in der Kategorie Algorithmen verfügbar. Hier können Sie festlegen, wie die Kanonisierung durchgeführt werden soll.

Standardmäßig wird eine Exclusive XML Canonicalization ausgeführt. Hier können Sie festlegen, dass stattdessen eine Inclusive XML Canonicalization ausgeführt werden soll.
