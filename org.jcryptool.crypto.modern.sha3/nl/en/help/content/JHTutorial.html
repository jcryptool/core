SHA-3 Candidate – JH

Description

JH is a very efficient algorithm. Its design is based on coherent round functions. A compression function is used to reduce the input out of a big permutation (bijective function). JH is applied on the general AES design SPN (substitution-permutation network) and MDS (maximum distance separable). JH uses an array with 8 dimensions. This design combines the characteristics of AES (SPN + MDS) and Serpent (SPN + bitslice).

The following bitlengths are supported:

  • 224 bit
  • 256 bit
  • 384 bit
  • 512 bit
