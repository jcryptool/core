What is JCrypTool?

JCrypTool (JCT) is a cryptography e-learning platform developed by the open-source project CrypTool. JCT is available at no costs, and runs under Linux, Windows and macOS. JCT's aim to interest people for cryptography and to offer them an easy-to-use and modern didactic tool.
JCT enables playful experiments with various cryptographic functions on arbitrary files. Users are mostly guided step-by-step through every process. The online help contains background information about the history or the technical details of algorithms and of their cryptanalysis.
More information on JCT's possibilities and their usage is available in the online help in the User Guide.


How can I extend JCrypTool with my own cryptographic plug-in?

JCrypTool (JCT) is written in Java and based on the Eclipse Rich Client Platform (RCP) and can therefore be extended at almost any place. In order to provide consistent and easy-to-use access to all functionality, JCT offers different extension points which must be used by all cryptographic plug-ins. This way, cryptography plug-ins can offer encryption algorithms, digital signatures, cryptanalysis or visualizations, which then appear at once in a common way like the ones in the Algorithm view.
More information on developing plug-ins is available in the JCT Wiki on GitHub.
