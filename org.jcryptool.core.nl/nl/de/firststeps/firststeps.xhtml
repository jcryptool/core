Perspektiven

JCrypTool stellt zwei Perspektiven zur Verfügung: Standard und Algorithmen. Dem Einsteiger bietet die Standard-Perspektive alle Möglichkeiten in der gewohnten Form von Menüs und Sichten an. Fortgeschrittene Anwender können in der Algorithmen-Perspektive moderne Verfahren in vielen Varianten nutzen. Der Wechsel zwischen den beiden Perspektiven erfolgt über die Ikonen rechts oben oder über das Menü Fenster -> Perspektive öffnen. Mehr...


Standard-Perspektive

Die Standard-Perspektive ist die Dokumenten- bzw. Datei-orientierte Hauptperspektive, die einen einfachen Zugriff auf alle wichtigen JCrypTool Funktionen ermöglicht. In dieser Perspektive können über den Datei-Explorer auf der linken Seite beliebige Dateien geöffnet werden. Über den Krypto-Explorer auf der rechten Seite können kryptografische Operationen aufgerufen werden.

Dokumenten-orientiert bedeutet, dass man erst eine Datei öffnet und dann auf dieser das gewünschte kryptografische Verfahren ausführt. Die Originaldatei bleibt dabei stets erhalten, jedes Verfahren generiert immer eine neue, temporäre, Datei.

Nicht alle kryptografischen Verfahren benötigen eine geöffnete Datei in einem der Editoren: Visualisierungen und Spiele werden beispielsweise in einer eigenem Sicht angezeigt. Mehr...


Algorithmen-Perspektive

Die Algorithmen-Perspektive bietet einen Funktions-orientierten Zugriff auf die in JCrypTool enthaltene Krypto-Bibliothek FlexiProvider. Die angebotenen Algorithmen passen sich dabei dynamisch dem Umfang der jeweiligen FlexiProvider-Version an, d.h. eine neue Version mit neuen Algorithmen führt automatisch zu neuen dynamischen Assistenten.
