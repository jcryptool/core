4. State diagram of the Number Shark program

The figure below shows the state diagram of the Number Shark program. Per default the maximum size of the number set is restricted to N=1024. The value is stored in a text file "nMax.txt". If you like to play on a larger number set, please exchange in this file the 1024 with a larger number.
Pay attention, for larger numbers the support feature as well as the calculation of optimal move sequences will be very time consuming.
