Datei-Explorer Benutzervorgaben

In den Benutzervorgaben für den Datei-Explorer können Sie Ihr Benutzerverzeichnis auswählen. Dieses wird immer beim Aufruf der Sicht expandiert angezeigt. Außerdem können Sie mit dem Home Icon schnell zu diesem Verzeichnis springen.

Die nächsten beiden Optionen sind vor allem beim Bearbeiten von großen Dateien nützlich. Normalerweise öffnet JCrypTool bei kryptografischen Operationen vor der Operation die Originaldatei in einem der Editoren. Nach der Operation wird die generierte Datei in einem neuen Editor angezeigt. Bei großen Dateien oder auf langsameren PCs kann das sehr viel Zeit benötigen. In diesem Fall können Sie festlegen, dass die Originaldatei und/ oder die generierte Datei nicht automatisch geöffnet werden sollen. Diese Einstellungen beeinflussen nur kryptografische Operationen, die über das Kontextmenü im Datei-Explorer aufgerufen werden.
