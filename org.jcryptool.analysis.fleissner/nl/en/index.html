Grille Cipher Analysis (Analysis, Encryption and Decryption)

This plugin allows to encrypt and decrypt text with the grille square, but above all you can also break grille ciphertexts automatically (both for even and odd square sizes). The maximum square size is 20 x 20.

Here you find help about the cipher itself and how to use this plug-in.

How does the algorithm work

A grille cipher is a technique for encrypting a plaintext by writing it onto a sheet of paper through a pierced sheet (of paper or cardboard or similar).

More details can be found in Wikipedia and in the online help of the pure visualization in JCT: Grille Cipher.

How to use this plug-in (instruction manual)

The PDF document Usage of the Grille Analysis explains in great detail how to use this plug-in. Each function is complemented with a screenshot.

Remark: JCT also contains an older plugin Grille Cipher, which is only intended to visualize the encryption and decryption process. It cannot break ciphertexts. Within the older plugin, the square size is limited to the values 4, 6, 8, and 10, while here this limit is abolished. If you want to learn more about this cipher, please also have a look at the online help of the older plugin.
