Tipps und Tricks


Maximimieren des Fensters eines Plugins

Falls Sie in JCT mehrere Fenster geöffnet haben, z.B. Editoren in der Standardansicht, die Kontexthilfe und einige Visualisierungen, und dann eines der Fenster maximieren wollen, doppelklicken Sie einfach den entsprechenden Reiter (bspw. auf den Reiter "Häufigkeits-Analyse" innerhalb der Visualisierungen).


JCrypTool im Netzwerk oder auf einem USB-Stick verwenden

Prinzipiell kann JCrypTool auch im Netzwerk oder auf einem externen Speichermedium wie einem USB-Stick oder einer USB-Festplatte verwendet werden. Dazu müssen Sie jedoch eine Änderung in einer Konfigurationsdatei vornehmen. Konkret müssen Sie den Pfad zum Workspace-Ordner anpassen. In diesem Ordner speichern alle Plug-ins die von Ihnen getroffenen Einstellungen.

Windows/ Linux:
Öffnen Sie die JCrypTool.ini in Ihrem Installationsordner und suchen Sie nach der Zeile -data @user.home/documents/.jcryptool. Hier müssen Sie @user.home/documents/.jcryptool durch einen in Ihrer Umgebung verfügbaren absoluten Pfad ersetzen.

macOS:
Die durchzuführende Änderung ist identisch wie unter Windows oder Linux. Allerdings befindet sich die JCrypTool.ini auf macOS unter JCrypTool.app\Contents\MacOS in Ihrem Installationsordner.


JCrypTool aktualisieren

Details dazu finden sich in Updates .

JCrypTool verfügt über einen integrierten Aktualisierungsmechanismus. Diesen können Sie über das Menü Hilfe aufrufen. Dort finden Sie die beiden Einträge Nach Aktualisierungen suchen und Neue Erweiterungen installieren.... JCrypTool sucht nicht automatisch regelmäßig nach Aktualisierungen. Sie müssen dies entweder manuell aufrufen oder in den Benutzervorgaben auf der Seite Automatische Aktualisierungen die entsprechenden Vorgaben festlegen.


Aktualisierungen werden nicht angezeigt

Es kann vorkommen, dass Aktualisierungen/Änderungen trotz erfolgreicher Installation nicht angezeigt werden. Das kann bei neuen Plug-ins und auch bei Aktualisierungen von vorhandenen Plug-ins der Fall sein. Ursache hierfür ist der von JCrypTool verwendete Workspace Ordner, in dem die Informationen über alle installierten Plug-ins gespeichert werden. Bei Änderungen kann es nun vorkommen, dass die dort gespeicherten Einstellungen nicht aktualisiert werden.

In diesem Fall können Sie entweder das .jcryptool Verzeichnis in Ihrem Dokumente Verzeichnis (innerhalb Ihres Benutzerverzeichnisses) von Hand löschen. Unter macOS und Linux ist dieser Ordner standardmäßig versteckt. Alternativ können Sie einmalig den Parameter -clean beim Aufruf von JCrypTool anhängen (z.B. über die Verknüpfung unter Windows). Hiermit wird der Workspace automatisch gelöscht und beim nächsten Start neu angelegt und initialisiert. Anschließend sollten Sie den Parameter wieder entfernen.

Hier sind zwei Beispiele unter Windows, wie der Kommandozeilen-Aufruf aussehen kann (legt man JCT auf eine Ikone im Startmenü sieht man den Befehl im Eigenschaften-Fenster der Ikone nach dem Feld "Ziel"):
F:\Downloads\jcryptool\JCrypTool.exe -clean
F:\Downloads\jcryptool\JCrypTool.exe -nl de -data %LOCALAPPDATA%\jct-de -clean

Und hier ein Beispielaufruf unter Linux:
$ ./JCrypTool -nl de -clean -data /home/Documents/.jcryptool/de &
