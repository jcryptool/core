Tipps und Tricks


Schneller Weg, einen Editor zu öffnen

Um einen Editor (Text oder Hex, leer oder gefüllt mit einer Standarddatei) zu öffnen, klicken Sie einfach auf die Ikone ganz links im Haupt-Toolbar oder den Drop-down-Pfeil rechts daneben. Screenshots mit wichtigen Ikonen finden Sie in der Hilfedatei Allgemeine Informationen.


Maximimieren des Fensters eines Plugins

Falls Sie in JCT mehrere Fenster geöffnet haben, z.B. Editoren in der Standardansicht, die Kontexthilfe und einige Visualisierungen, und dann eines der Fenster maximieren wollen, doppelklicken Sie einfach den entsprechenden Reiter (bspw. auf den Reiter "Häufigkeits-Analyse" innerhalb der Visualisierungen).


Wie findet man in JCT schnell Infomationen?

Zuallererst können Sie die Lupen-Ikone im Haupt-Toolbar anklicken, um Hilfe zu einem bestimmten Thema zu bekommen. Dann öffnet sich das Quick-Access-Suchfeld: Suchen Sie darin am besten nach dem Namen eines Plug-ins oder Verfahrens. Screenshots mit wichtigen Ikonen finden Sie in der Hilfedatei Allgemeine Informationen.

Alternativ können Sie die ausführliche Onlinehilfe in einem Browser öffnen: Klicken Sie dazu in JCT die Fragezeichen-Ikone im Haupt-Toolbar. In der Onlinehilfe klicken Sie dann auf der linken Seite auf den Eintrag Anwenderhandbuch. Jetzt sehen Sie die verschiedenen Kategorien, in denen die Hilfe in JCT kategorisiert ist. Diese Kategorien können Sie mit einem Klick auf die kleine Plus-Ikone vor dem jeweiligen Eintrag öffnen. So können Sie sich bis zur gesuchten Hilfeseite durchnavigieren und bekommen einen guten Überblick. Alternativ können Sie auch das Suchfeld (oben unter der Browser-Toolbar) in der Onlinehilfe nutzen.


JCrypTool aktualisieren

JCrypTool verfügt über einen integrierten Aktualisierungsmechanismus. Details dazu und Hinweise zu Problembehebungen dabei finden sich in Updates.


JCrypTool im Netzwerk oder auf einem USB-Stick verwenden

Prinzipiell kann JCrypTool auch im Netzwerk oder auf einem externen Speichermedium wie einem USB-Stick oder einer USB-Festplatte verwendet werden. Dazu müssen Sie jedoch eine Änderung in einer Konfigurationsdatei vornehmen. Konkret müssen Sie den Pfad zum Workspace-Ordner anpassen. In diesem Ordner speichern alle Plug-ins die von Ihnen getroffenen Einstellungen.

Windows/ Linux:
Öffnen Sie die JCrypTool.ini in Ihrem Installationsordner und suchen Sie nach der Zeile -data @user.home/documents/.jcryptool. Hier müssen Sie @user.home/documents/.jcryptool durch einen in Ihrer Umgebung verfügbaren absoluten Pfad ersetzen.

macOS:
Die durchzuführende Änderung ist identisch wie unter Windows oder Linux. Allerdings befindet sich die JCrypTool.ini auf macOS unter JCrypTool.app\Contents\MacOS in Ihrem Installationsordner.
