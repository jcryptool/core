Perspectives

Perspectives can be regarded as containers, in which other components, namely views and editors, are embedded and can be layouted in almost any way you desire.

JCrypTool offers two perspectives: Default and Algorithm. The Default Perspective shows all possibilities in a usual way – with menus and views for the beginner. The Algorithm Perspective is more suitable for advanced users and offers many variants of the cryptographic operations.

Initially, JCrypTool offers a pre-defined layout of both perspectives. In case of mis-configuration or other problems you can use the menu Window -> Reset Perspective to go back to the initial configuration of that perspective.

After launching JCrypTool, always the last active perspective is shown. In order to switch between the existing perspectives, use the buttons Default and Algorithm in the upper right of the main window, or the menu Window -> Open Perspective. The menu Window -> Customize Perspective opens a dialog in which you can select the elements (menu entries and toolbar icons) that should be displayed in the current perspective.

Views in different perspectives are independent of each other. Closing or moving a view in the Default Perspective has absolutely no effect on the same view in the Algorithm Perspective. On the other side, editors are shared between perspectives. Closing an editor in the Default Perspective closes this editor in the Algorithm Perspective as well.
