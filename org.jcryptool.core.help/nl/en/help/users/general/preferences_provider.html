Crypto Provider

The preferences dialog contains an overview about all in JCrypTool available crypto providers and their versions. With the Up and Down buttons (on the right side) it is possible to select their order. The first crypto provider automatically becomes the default crypto provider.

When selecting a certain algorithm in one of the crypto plug-ins, JCrypTool tries to find an implementation for it. The installed crypto providers are searched through whether they provide this algorithm. The provider found first is used. In case none of the installed providers offers support for an algorithm an error is shown.

Currently the default crypto provider in JCrypTool is FlexiProvider. Additionally BouncyCastle is included in every installation. BouncyCastle will become the default crypto provider in one of the next versions of JCT.
