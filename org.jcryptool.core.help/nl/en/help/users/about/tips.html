Tips and Tricks


Quick way to open a new editor

To open a new editor (text or hex, empty or filled with a default file) just click on the leftmost icon in the main toolbar or the drop-down arrow next to the right. See the screenshots with important icons in the help file General Information.


Maximizing the window of a plugin

You might have opened the default view of JCrypTool (JCT) with editors, the context help, and some visualizations. If you then want one of them to be maximized just double-click on the tab (e.g. on the "Frequency Analysis" tab within the visualizations).


How to quickly find information in JCrypTool

First of all, you can click the magnifying glass icon in the main toolbar in order to get help about a particular topic. Then the Quick Access search field (Ctrl-3) opens: Preferably, you can look for the name of plug-ins or of methods (if looking for strings in the online help click the according item which opens the list in another browser tab). The search filter reacts at once, even before the term in typed in completely. Maybe you want to try with "Vig" or "Elie".

Screenshots with important icons in the help file General Information.

Alternatively, you can open the exhaustive online help in a browser: To do so, click in JCT on the question-mark icon within the main toolbar. Within the online help then click on the left on the entry User Guide. Now you can see the different categories, used to structure the help in JCT. To open a category, please click on the small plus icon in front of the particular category. So you can navigate to the according help page and get a good overview. Alternatively, you can use the search field (at the top below the browser toolbar) in the online help.


Changing the language in JCrypTool

In the general preferences of JCrypTool (JCT) you can change the language. After a change JCT asks you to automatically restart with the new selected language. Further details about this can be found in Changing the Language.


Updating JCrypTool

JCrypTool contains an integrated update mechanism. Details about this and hints for problem solving can be found in Updates.


Using JCrypTool in a network environment or on a USB stick

In principle it is possible to use JCrypTool in a network environment or on an external storage medium like a USB stick or a USB hard disk. To enable this you have to do a minor change in one of the configuration files: It is necessary to change the path to the default workspace folder. This folder is used by all plug-ins to store your specific preferences.

Windows/ Linux:

Open the file JCrypTool.ini in your installation folder and search for the line -data @user.home/documents/.jcryptool. Replace @user.home/documents/.jcryptool with the absolute path to a folder available in your environment.

macOS:

The required changes are identical with the ones described above for Windows/Linux. However, the file JCrypTool.ini is located at JCrypTool.app\Contents\MacOS in your installation folder.
