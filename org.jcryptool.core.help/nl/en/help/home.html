JCrypTool Online Help

Welcome to the JCrypTool online help!

From now on we use the abbreviation JCT for JCrypTool.

The User Guide contains an introduction to JCT. It describes how to use JCT and what functions the various cryptographic plug-ins offer, e.g. for encryption or digital signatures. This is the best entry point if you have never used JCT before.

An exhaustive overview what you can do with JCT can be found in the User presentation of the JCrypTool project.

Developers find relevant and up-to-date information in our Wiki.

More information about JCT is available on our home page and in the help file About JCrypTool.
